# Drug Inventory and Supply Chain Management System

## 📌 Project Overview
This project focuses on analyzing drug inventory data to identify shortages, optimize stock levels, and improve supply chain efficiency.

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- SQL
- Power BI
- Excel

## 📊 Key Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Inventory trend analysis
- Interactive dashboard using Power BI

## 📈 Outcome
- Improved inventory tracking
- Identified stock shortages and patterns
- Reduced chances of overstock and wastage
